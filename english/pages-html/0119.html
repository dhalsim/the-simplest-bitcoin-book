Dispelling FUD
BITCOIN IS A PONZI
Bitcoin is not a Ponzi:
Old investors are not paid any money by new investors.
When buying bitcoin, no one is promising a return on your investment.
There is no leader or promotional team.
No mining has been done before.
Read: Lyn Alden's 'Why Bitcoin is Not a Ponzi Scheme' article.
BITCOIN IS TOO SLOW
Although the Bitcoin network is slow, it is built on top of a second layer called the Lightning Network, which is much faster.
The Bitcoin network can process around 7 transactions per second (TPS).
Visa network claims to process up to 24,000 transactions per second, but real-world usage is closer to 4,000 TPS.
The Lightning Network can process millions of transactions per second!
