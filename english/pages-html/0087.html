How Does Work?
GLOSSARY OF TERMS
TRANSACTION ~ Sending/receiving bitcoin
A transfer of value in the form of satoshis, from one bitcoin holder to another.
NODE ~ A 'branch' of the decentralized bitcoin 'bank'. Anyone can run a node.
A computer connected to the Bitcoin network.
MINING ~ The process of verifying transactions and adding them to the blockchain
Nodes are computers running the Bitcoin software.
Nodes work together to maintain the integrity of the blockchain.
Each full node contains a record of all transactions and balances.
Nodes verify transactions and add them to the blockchain.
Nodes are run by volunteers around the world.
Full nodes have been running since 2009, when Bitcoin was created.
The more nodes there are, the more decentralized the network becomes.
There are currently over 19,000 full nodes and many more unverified nodes.
All nodes are equal.
